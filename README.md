# Innotec HTML Hub

## 🇩🇪 Deutsch

Der **Innotec HTML Hub** ist eine zentrale Übersicht für interne HTML-Seiten wie:

- Kalkulatoren
- Produktseiten
- Tools und Prototypen

Alle Seiten werden über eine gemeinsame Startseite bereitgestellt und können direkt im Browser geöffnet werden.

---

### 🔧 Funktionsumfang

- Übersicht aller HTML-Seiten in Kartenansicht
- Kategorien & Tabs (z. B. Material, Vergleich, Szenario-Rechner)
- Suchfunktion
- Sprachumschaltung (DE / EN)
- Zentrale Pflege über `pages.json`
- Hosting über GitHub Pages

---

### 📄 Aktuell enthaltene Tools

- Mehrwert-Berechnung Additive
- Materialrechner PV
- Bindemittel Vergleich
- Materialrechner Projekt
- Bindemittel Kostenvergleich
